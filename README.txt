ЧИТАЙ ПО СЛОГАМ — ВЕРСИЯ НА 30 СЛОВ

Откройте index.html в браузере.

Порядок работы:
1. Нажмите слоги по порядку.
2. Для записанных слогов звучит WAV.
3. После последнего слога звучит целое слово MP3.
4. Затем звучит случайная похвала.
5. Автоматически открывается следующее слово.

В проекте подключены все 30 записей слов и 30 изображений.
Пока записаны 14 отдельных слогов; для остальных кнопка работает без звука, после чего цепочка продолжается.


ИСПРАВЛЕНИЯ ВЕРСИИ 2:
- Блок «Ты молодчинка!» теперь является кнопкой и проигрывает случайную похвалу.
- После завершения слова появляется заполненная жёлтая звезда.
- Подсветка текущего слова выровнена строго по строке.
- Все изображения пересобраны в точном размере окна и усилены по резкости.
