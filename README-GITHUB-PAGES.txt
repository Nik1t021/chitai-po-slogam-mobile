ЧИТАЙ ПО СЛОГАМ — ВЕРСИЯ ДЛЯ GITHUB PAGES И IPHONE

Эта папка уже подготовлена для GitHub Pages, Safari и добавления на экран «Домой».
Никакие приложения из App Store, профили или дополнительные программы на iPhone устанавливать не нужно.

КАК ОПУБЛИКОВАТЬ
1. Создайте пустой репозиторий GitHub.
2. Загрузите В САМ КОРЕНЬ репозитория всё содержимое этой папки:
   index.html, app.js, data.js, style.css, manifest.webmanifest, service-worker.js,
   .nojekyll, а также папки audio, images и icons.
3. В репозитории откройте Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main, Folder: / (root), затем Save.
6. Откройте выданную GitHub Pages ссылку в Safari.

КАК ДОБАВИТЬ ИКОНКУ НА ЭКРАН «ДОМОЙ»
1. Откройте сайт в Safari.
2. Нажмите «Поделиться».
3. Нажмите «На экран Домой».
4. Нажмите «Добавить».

Это не установка из App Store. На главном экране появится обычная иконка,
а приложение будет открываться отдельно от вкладок Safari.

ПЕРВЫЙ ЗАПУСК
- Нужен интернет, чтобы Safari один раз сохранил картинки и звуки.
- После завершения кэширования приложение сможет открываться и без интернета.
- Звук активируется первым нажатием ребёнка на слог, слово или похвалу.

ВАЖНО
- GitHub Pages должен работать по HTTPS (обычно GitHub включает его автоматически).
- Не переименовывайте файлы и папки: пути к аудио и изображениям уже настроены.


МОБИЛЬНАЯ ВЕРСИЯ
На экранах шириной до 820px автоматически включается отдельная вертикальная раскладка.
Список слов прокручивается горизонтально, картинка и слоги занимают всю ширину экрана.
